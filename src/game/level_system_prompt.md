# 레벨업 시스템 설계

## 기본 정보
- 총 레벨: 150
- 무기 슬롯: 3개, 각 무기당 속성 3칸 = 총 9칸
- 속성: 흙, 불, 물, 전기, 빛, 암흑 (6종, 중복 허용)
- Lv.1~9: 기본 공격만 가능 (속성 무기 없음)

## 속성 획득 (10레벨 단위 이벤트)

### 속성 선택 (Lv.10, 20, 30)
- Lv.10: 랜덤 속성 후보 중 택 1 → 무기1 첫번째 칸 고정 배치
- Lv.20: 랜덤 속성 후보 중 택 1 → 무기2 첫번째 칸 고정 배치
- Lv.30: 랜덤 속성 후보 중 택 1 → 무기3 첫번째 칸 고정 배치

### 포커 대결 (Lv.40, 50, 60, 70, 80, 90)
- 텍사스 홀덤 간소화 방식으로 딜러 AI와 대결
- 승리: 속성 자유 선택 + 빈 슬롯 자유 배치
- 폴드: 랜덤 속성 + 랜덤 빈 슬롯에 배치
- 패배: 속성 획득 없음

### 구제 포커 (Lv.100, 110)
- 패배로 빈 슬롯이 남아있는 경우에만 활성화
- 규칙은 포커 대결과 동일
- 빈 슬롯이 없으면 스킵 → 스탯 강화로 대체

## 일반 레벨업 (10단위 이벤트 외 전부)
- 3~4개 선택지 중 택 1
- 스탯 강화: 데미지 증가, 공격 속도, 체력, 이동 속도, 크리티컬, 경험치 보너스 등
- 9칸 모두 채워진 이후 (최소 Lv.91~) 레벨업은 전부 스탯 강화

## 핵심 밸런스
- 포커 전승 시 Lv.90에 9칸 완성 → Lv.91~150 전부 스탯 강화 (60레벨 분량)
- 구제까지 간 경우 완성이 늦어진 만큼 스탯 강화 구간이 줄어듦 → 후반 화력 열세
- Lv.110 이후에도 빈 칸이 있으면 미완성 상태로 런 진행
